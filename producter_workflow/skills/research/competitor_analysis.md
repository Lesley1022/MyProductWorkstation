# Skill: competitor_analysis

## 所属Agent
调研分析Agent (Research Agent)

## 描述
生成竞品分析文档，对比主要竞争对手的产品、市场定位、优劣势、策略等。帮助明确产品的差异化优势。

## 输入参数
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| `project_id` | string | 是 | 项目标识 |
| `platform_id` | string | 是 | 平台标识 |
| `version` | string | 是 | 版本号 |
| `common_rules` | object | 否 | DAO OS通用规则文档 |
| `previous_docs` | array | 是 | 已生成的文档列表，至少包含需求收集、市场分析 |
| `pending_confirms` | array | 是 | 待确认项列表 |

## 输出参数
| 参数名 | 类型 | 描述 |
|--------|------|------|
| `document_content` | string | 按照“竞品分析模板”生成的完整文档内容 |

## 内部逻辑
1. 加载“竞品分析模板”。
2. 基于市场分析和行业知识库，识别3-5个主要竞品（如国内主流虚拟电厂平台）。
3. 对每个竞品，从以下维度进行分析：
   - 产品概述与核心功能
   - 目标用户与市场定位
   - 技术架构与集成能力
   - 商业模式与定价
   - 优劣势总结
4. 制作对比表格，突出本项目的潜在差异化点。
5. 返回生成的文档内容。

## 注意事项
- 竞品信息应基于公开资料，若信息不足需注明。
- 分析需客观，避免主观评价。
