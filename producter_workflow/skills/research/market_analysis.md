# Skill: market_analysis

## 所属Agent
调研分析Agent (Research Agent)

## 描述
生成市场分析文档，包含市场规模、市场趋势、竞争格局、市场机会等。该技能基于项目背景和已有文档（如需求收集）进行分析，并结合行业知识库提供专业洞察。

## 输入参数
| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| `project_id` | string | 是 | 项目标识 |
| `platform_id` | string | 是 | 平台标识 |
| `version` | string | 是 | 版本号 |
| `common_rules` | object | 否 | DAO OS通用规则文档 |
| `previous_docs` | array | 是 | 已生成的文档列表，至少包含需求收集文档 |
| `pending_confirms` | array | 是 | 待确认项列表，可能影响分析方向 |

## 输出参数
| 参数名 | 类型 | 描述 |
|--------|------|------|
| `document_content` | string | 按照“市场分析模板”生成的完整文档内容 |

## 内部逻辑
1. 加载“市场分析模板”。
2. 从 `previous_docs` 中提取需求收集文档中的业务目标、目标用户等信息。
3. 结合虚拟电厂行业知识库（包括国内外市场数据、典型玩家、政策导向），调用AI模型撰写以下章节：
   - 市场规模与增长趋势
   - 市场驱动因素与阻碍因素
   - 主要竞争对手及市场份额
   - 目标细分市场分析
   - 市场机会点
4. 确保内容与需求收集文档中的业务目标一致，若存在待确认项，在文档中注明“基于当前假设”。
5. 返回生成的文档内容。

## 注意事项
- 市场数据应注明来源或假设，如“基于XX报告预测”。
- 若需求收集文档中存在待确认项，应在分析中保持灵活性，避免过度依赖未确认信息。
