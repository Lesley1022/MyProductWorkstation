# DAO OS 通用设计规范

本规则为 DAO OS 系统的通用设计规范。如各功能模块有特殊规则，应在对应模块的详细设计文档中明确说明；如无特殊说明，则一律遵循本通用规则。

## 查询条件

| 字段类型 | 说明 |
| :--- | :--- |
| 文本框 | 1. 最多支持输入50个字符（1个汉字、数字、符号均计为1个字符）<br>2. 支持对输入内容进行模糊查询 |
| 下拉框（下拉单选框/下拉多选框） | 1. 下拉展示所有选项，支持对下拉选项进行模糊查询<br>2. 当页面存在多个独立的下拉框时，默认至少有一个下拉框需选中选项 |
| 区域选择框 | 下拉展示所有选项，支持对下拉选项进行模糊查询 |
| 日期区间选择框 | 1. 格式为 YYYY-MM-DD<br>2. 时间起 ≤ 时间止 ≤ 当前系统时间<br>3. 默认日期区间为当前系统时间所在年月的1号到当前系统时间所在年月日<br>4. 起止时间是否允许跨月选择应根据业务场景配置，默认可跨月。若需限制，请在具体功能说明中明确 |
| 日期选择框 | 1. 格式为 YYYY-MM-DD<br>2. 所选日期 ≤ 当前系统时间<br>3. 默认选中当前系统时间所在年月日<br>4. 支持日期的前后快速切换 |
| 提示 | 1. 若未查询到，则提示：暂无符合查询条件的数据<br>2. 若查询时间超过3秒，则提示：正在查询中，数据量较大，请稍等！<br>3. 若系统错误，则提示：系统繁忙，请稍后再试！<br>4. 若网络异常：则提示：网络异常，请检查网络后重试！ |
| 提示样式 | 1. 未查询到：展示在空白的列表界面中央<br>2. 查询时间较长：以进度条形式显示在对应列表界面中央，直到查询出对应数据<br>3. 系统错误：展示在空白的列表界面中央<br>4. 网络异常：展示在空白的列表界面中央 |
| 查询条件 | 支持对选中的查询条件进行删除重选 |
| 查询 | 默认必须有 |
| 重置 | 默认必须有，重置为默认查询条件 |

## 功能按钮

| 功能按钮 | 操作说明 | 校验 | 提示 |
| :--- | :--- | :--- | :--- |
| 导入 | 点击后弹框提示：<br>1. 单个导入文件 ≤5M（自定义）<br>2. 仅支持导入 xlsx、xls（自定义）格式文件<br>3. 导入为增量导入，即在现有列表数据基础上新增<br>注意：<br>1) 弹框点击确定后执行导入校验操作，点击取消则关闭弹框，不执行导入校验操作，点击不再提示，则该账号后续登录点击该按钮后不再提示<br>2) 默认一次仅支持选择一个文件导入（可配置） | 1. 校验导入文件格式、列名、列顺序与模板是否一致<br>2. 校验必填项<br>3. 校验填写的字段值是否符合要求<br>4. 校验唯一性<br>注意：<br>1) 校验到第一个不符合内容则进行提示，并不再往后进行校验<br>2) 导入内容都符合校验规则则全部导入，否则不导入 | 1. 导入文件与模板不符，请重新选择导入文件！<br>2. 第【行数】行的【字段名称】字段未填写，请填写后重新导入！<br>3. 第【行数】行的【字段名称】字段未正确填写，请规范填写后重新导入！<br>4. 第【行数】行的【校验唯一性字段值】已存在，不可重复导入，请删除后重新导入！ |
| 导出 | 1. 若有模板则按照模板进行导出<br>2. 若无模板则按照选择字段进行导出（除操作列）<br>3. 若未勾选记录，则导出当前查询条件下的全部记录<br>4. 若勾选了则仅导出勾选记录<br>5. 通过浏览器下载方式导出<br>6. 导出文件名默认为自定义文件名_导出日期（YYYYMMDD）<br>7. 默认必须有该按钮<br>8. 若导出的是 excel 文件，则导出的文件中的字段数据类型需与列表界面字段数据类型一致，文件表头加粗、居中，字体均为宋体11号 | | |
| 新增 | 点击后跳转或打开对应新增界面 | | |

## 列表

| 字段名称 | 说明 |
| :--- | :--- |
| 序号 | 从1开始自增（包含翻页），固定展示列 |
| 字段值为空 | 显示为 -- |
| 列表排序 | 支持点击数值型、日期时间型、标准状态或类型的表头，对列进行升序/降序排序 |
| 表格间距 | 可拖动表格间距 |
| 自定义字段 | 可自定义展示表头字段 |
| 表头固定 | 表头字段固定展示 |
| 操作 | 固定展示列，最多展示两个高频操作功能按钮，超过部分放置在更多按钮内，点击下拉展示。只展示当前列可操作的按钮。确定和取消按钮的组合和排列形式，确定在右边，取消在左边 |
| 编辑 | 仅可对保存成功但未提交的列表数据进行该操作，点击后打开对应保存成功界面，可对界面内容进行编辑、修改操作 |
| 修改 | 可对提交成功的列表数据进行该操作，点击后打开对应提交成功界面，可对界面内容进行修改操作 |
| 详情 | 可对列表数据进行查看详情操作，点击后打开对应操作成功的界面，仅可进行查看操作 |
| 审批 | 可对待审批状态的列表数据进行审批操作，点击后打开对应审批的界面，可选择审批结果、填写审批原因 |
| 启用 | 停用状态支持启用操作，启用操作前需弹窗确认【您确认要启用吗？启用后该数据将恢复使用，请确认！】，启用后数据生效 |
| 停用 | 启用状态支持停用操作，停用操作前需弹窗确认【您确认要停用吗？停用后该数据将不允许使用，请确认！】，启用后数据禁用。存在停用前是否有进行应用的判断和提示 |
| 删除 | 提交成功的数据不允许删除。删除操作前需弹窗确认【您确认要删除吗？删除后数据不可恢复，请谨慎操作！】，删除后数据不可恢复。不支持批量删除 |
| 注意 | 列表表头及对应字段值应完全展示，避免因宽度不足而被截断。存在表头过长的场景，支持存在横向滚动条 |
| 分页 | 列表默认支持分页，每页显示10条记录，用户可选择每页显示10/15/20/50/100条 |
| 数据权限 | 所有数据查询与操作均受角色与组织架构权限控制，用户仅能查看和操作其权限范围内的数据 |
| 操作提示 | 任何成功操作（删除、启用、停用等）后，应在界面给出明确的成功 toast 提示。如：删除成功！ |

## 具体界面

| 字段类型 | 说明 |
| :--- | :--- |
| 文本框 | 1. 最多支持输入50个字符（1个汉字、数字、符号均计为1个字符）<br>2. 默认对输入的文本去掉前后空格、全角空格、换行符、制表符、其他 Unicode 空白字符等 |
| 下拉框（下拉单选框、下拉多选框） | 下拉展示所有选项，支持对下拉选项进行模糊查询；支持一键清空 |
| 单选框 | 平铺展示所有选项 |
| 复选框 | 平铺展示所有选项；支持全选 |
| 日期区间选择框 | 1. 格式为 YYYY-MM-DD<br>2. 时间起 ≤ 时间止<br>3. 提供快捷操作 |
| 日期选择框 | 格式为 YYYY-MM-DD；支持左右快速切换 |
| 金额 | 单位默认为元，保留两位小数 |
| 手机号 | 仅支持输入11位数字 |
| 邮箱 | 必须包含 . 及 @ |
| 文本域 | 最多支持输入200个字符（1个汉字、数字、符号均计为1个字符） |
| 列表（明细行） | 若是必填，则至少填写一行有效数据 |
| 图片附件 | 支持在系统配置中自定义文件类型、数量、大小等限制，以下为默认配置：<br>1. 支持上传 jpg、png 格式文件（自定义）<br>2. 支持上传1个文件（自定义）<br>3. 单个文件 ≤1M（自定义） |
| 文件附件 | 支持在系统配置中自定义文件类型、数量、大小等限制，以下为默认配置：<br>1. 支持上传 doc、docx、xls、xlsx、pdf、txt、zip、rar 格式文件（自定义）<br>2. 支持上传1个文件（自定义）<br>3. 单个文件 ≤10M（自定义） |
| 保存 | 仅对填写了的数据做校验，保存过程中需显示进度条，并锁定界面（该界面不可进行任何操作），直到保存成功后或校验不通过<br>1. 校验填写的字段值是否符合要求：若不符合，则对应字段边框标红，并在边框下面提示：请正确填写<br>2. 校验唯一性：若数据重复，则弹框提示：【校验唯一性字段值】已存在，请勿重复填写！点击关闭按钮后关闭提示<br>3. 保存成功后，需 toast 提示：保存成功！<br>4. 若系统错误，则提示：系统繁忙，请稍后再试！<br>5. 若网络异常：则提示：网络异常，请检查网络后重试！ |
| 提交 | 提交过程中需显示进度条，并锁定界面（该界面不可进行任何操作），直到提交成功后或校验不通过<br>1. 校验必填项：若未填写，则对应字段边框标红，并在边框下面提示：请填写<br>2. 校验填写的字段值是否符合要求：若不符合，则对应字段边框标红，并在边框下面提示：请正确填写<br>3. 校验唯一性：若数据重复，则弹框提示：【校验唯一性字段值】已存在，请勿重复提交！点击关闭按钮后关闭提示<br>4. 提交成功后，需 toast 提示：提交成功！<br>5. 若系统错误，则提示：系统繁忙，请稍后再试！<br>6. 若网络异常：则提示：网络异常，请检查网络后重试！ |

## 数据脱敏规则

列表或界面支持查看明文操作，但仅相关角色（看具体配置）可显示及操作，为保障用户信息安全性，不应只在界面做脱敏，而应在存储层脱敏（如用哈希、掩码、加密）。

| 字段名称 | 脱敏规则 |
| :--- | :--- |
| 证件号码 | 显示第一位和最后一位，如：3************4 |
| 银行卡号 | 显示后四位，如：***********2345 |
| 手机号码 | 显示前三后四位，如：139****8978 |
| 电子邮箱 | 显示第一位和@及后面的内容，如：z***@example.com |
