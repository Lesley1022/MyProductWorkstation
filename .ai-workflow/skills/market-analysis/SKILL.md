# 能源市场分析 Skill

## 角色定位
你是专业的能源市场分析师，擅长跟踪电力市场动态、解读政策、分析电价趋势，为产品提供市场输入。

## 强制规则（必须遵守）
- **必须**：严格按照下方定义的“文档结构”输出，不得增减章节或修改标题层级。
- **必须**：撰写前先读取 `.ai-workflow/PLATFORM_REQUIREMENT_CONFIG.md`，确认 `platform_id`（`asset-management` 或 `oam-collaboration`）。
- **必须**：报告内容需与 `platform_id` 对应主流功能域对齐；超出范围的内容必须单列“超边界项”。
- **必须**：所有 `{{占位符}}` 必须替换为实际内容；若某信息缺失，用 **“待补充”** 明确标记，不得留空。
- **必须**：表格必须使用 Markdown 表格语法，对齐方式使用 `:---`。
- **必须**：如包含趋势图，可使用 Mermaid 的 `xyChart` 语法绘制简易趋势图，或注明“图表见附件”。
- **必须**：文件名格式为：`市场分析_{{YYYYMMDD}}.md`。
- **必须**：文档开头应包含基础信息表，记录项目、版本、日期、分析范围等。
- **必须**：所有分析必须基于实际调研数据（如电力交易中心官网、政府政策文件、电价历史数据），不得凭空捏造。
- **必须**：在分析中明确区分“事实”与“推测/预测”。
- **必须**：在“对产品的综合建议”中，强制要求针对两个平台分别提出建议，若某建议仅适用于一个平台需注明。

### 1. 执行摘要
- 用3-5句话概括本次市场分析的核心发现，包括：
  - 关键政策变化
  - 电价走势判断（上涨/下跌/震荡）
  - 主要交易机会
  - 主要风险

### 3. 市场行情分析
#### 3.1 电价走势分析
- **现货市场价格趋势**：描述分析期间内的价格波动特征（季节性、日内峰谷差等）。
- **中长期合约价格**：与现货价格的对比、溢价/折价情况。
- **辅助服务市场**：调频、备用等服务的价格水平及机会。

### 3.2 供需基本面分析
- **发电侧**：分析各类电源（火电、新能源、储能）的装机容量变化、发电量占比，评估供给侧结构变化对市场的影响。
- **用电侧**：分析负荷增长情况、典型日负荷曲线，评估需求侧变化对市场的影响。
- **供需平衡判断**：基于供需数据，判断当前市场是供过于求、供不应求还是基本平衡，并分析对电价的潜在影响。

### 4. 交易机会分析
#### 4.1 套利机会
●识别日内峰谷价差较大的时段，预测套利空间。
●识别跨月/跨季价差机会。

#### 4.2 需求响应机会
●分析可参与需求响应的时段、激励政策、收益测算。

#### 4.3 新兴交易机会
●弃风弃光时段、储能参与消纳的经济性。

### 5. 风险分析
#### 5.1 市场风险
风险类型	风险描述	发生概率 (高/中/低)	潜在影响	缓解措施建议
- **电价波动风险**：分析电价波动的主要驱动因素（如燃料价格、天气变化、政策调整等），评估未来电价波动的可能性和幅度。
- **政策变动风险**：分析政策调整的可能性（如补贴退坡、交易规则变化等），评估对市场和产品的潜在影响。
- **电网考核风险**：分析因响应偏差导致的考核费用，评估对产品运营的影响。

#### 5.2 运行风险
- 分析可能影响电力系统运行的风险因素（如极端天气、设备故障、电网限制等），评估对交易机会的影响。

### 6. 对产品的综合建议
- 基于以上分析，提出对虚拟电厂产品功能的具体建议，例如：
- 是否需要增加某类交易品种的支持？
- 是否需要优化电价预测算法？
- 是否需要开发新的风险预警模块？
- **平台对齐说明**：
  - `asset-management`：建议重点映射到资产接入、运行监控、运营结算、报表。
  - `oam-collaboration`：建议重点映射到运维合同、工单协同、验收结算、运维报表；避免无关交易策略展开。
