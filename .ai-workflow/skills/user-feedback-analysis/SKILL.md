# 用户反馈分析 Skill

## 结构要求
1. **数据来源**：反馈来自何处（客服记录、社交媒体、问卷调查）
2. **高频问题**：用词云或列表展示用户最关心的问题，建议构建电力行业关键词库（如“SOC不准”、“工单没人接”、“结算对不上”），自动聚类反馈。
3. **用户画像片段**：从反馈中提炼典型用户特征，区分资产方和运维方的反馈，分别提炼诉求。
4. **核心诉求**：用户真正想要的功能或改进，强调“结算准确”、“工单响应快”、“报表清晰”等行业核心需求。
5. **情感倾向**：用户对当前产品的满意度

## 强制规则
- 必须使用行业关键词库辅助聚类。
- 必须区分资产方和运维方反馈。
