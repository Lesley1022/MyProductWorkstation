# 用户操作手册 Skill

请遵循以下结构生成手册：

1.  **功能简介**：用一两句话介绍这个功能是干什么的。
2.  **适用人群**：哪些角色可以使用此功能（可细化电力行业角色）。
3.  **操作步骤**：
    *   步骤1：截图 + 文字说明（因无法直接截图，用文字详细描述点击路径和界面状态）。对于现场运维场景，需考虑移动端操作步骤。
    *   步骤2：……
4.  **常见问题 (FAQ)**：预测用户可能遇到的 2-3 个问题并给出解答，例如：
    *   “为什么结算金额与我手工计算的不一致？”
    *   “如何导出符合电网要求的报表？”
    *   “工单状态一直显示处理中怎么办？”
